Tool/Process Notes
DITA DTD version 1.1  
Open Toolkit version 1.5  
Customizations dita2xhtml Extensive overrides of dita2xhtml files
CSS Override of default CSS and new classes added via dita2xhtml files
Plug-in Dita4Publishers - epub 2009 dita4publishers.sourceforge.net
DITA XML Editors SynCro Soft Oxygen 11.1 Enterprise The editors are used to write and validate topics but not for generating output; generating output via the editor is an option to be checked.
Notepad, Wordpad
Generation of output Open Toolkit Output created with Open Toolkit procedures, Ant builds called via Command Prompt (the black window) or Java commands.
Source language English (en) The first language is written in one or a combination of XML editors
Target languages Dutch (nl) Translated from the English by a human translator using translation memory and terminology tools, with validation in the translation tool
  Spanish (es) XML files were machine-translated for a gist translation (speed more important than linguistic quality) and edited for validation (not for translation quality) in one of the XML editors
  Chinese (zh)  
Translation tools SDL Trados This is a Dita-aware translation memory tool that supports the DITA DTD.
